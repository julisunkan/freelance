Build a full-stack web application using Python Flask called:

"Online ID Validator (AI-assisted)"

# CORE PURPOSE
The app allows users to upload identity documents (front ID, back ID, optional selfie) and performs:
- OCR extraction of text
- Structured field extraction
- Image quality analysis
- Duplicate detection
- Risk scoring (LOW / MEDIUM / HIGH)
- AI-powered validation and correction using Groq API

The system is NOT a login-based app. It has:
- Public upload interface
- Hidden admin panel at /julisunkan

---

# TECH STACK
- Python Flask
- SQLite (SQLAlchemy)
- OpenCV (image analysis)
- pytesseract (OCR)
- PIL (image handling)
- Groq AI API (LLaMA 3 models)
- face_recognition (optional face match)
- hashlib (duplicate detection)
- HTML/CSS/JS frontend
- PWA support (manifest + service worker)

---

# MAIN FEATURES

## 1. USER UPLOAD SYSTEM
- Upload:
  - Front ID image (required)
  - Back ID image (optional)
  - Selfie image (optional)
- Store files in /uploads
- Save metadata in SQLite

---

## 2. OCR PROCESSING
- Extract text using pytesseract
- Clean OCR output
- Store raw text in DB

---

## 3. STRUCTURED DATA EXTRACTION
Extract:
- full_name
- date_of_birth
- id_number
- expiry_date

Use regex fallback if AI is unavailable.

---

## 4. IMAGE QUALITY ANALYSIS
Use OpenCV to detect:
- blur level
- low resolution
- image noise
Return an image quality score.

---

## 5. DUPLICATE DETECTION
- Hash uploaded images using SHA256
- Detect repeated uploads

---

## 6. RISK SCORING ENGINE
Compute risk level based on:
- image quality
- missing fields
- OCR errors
- face mismatch (if selfie provided)

Output:
- LOW
- MEDIUM
- HIGH

---

## 7. GROQ AI INTEGRATION (IMPORTANT)

Admin can set Groq API key in settings page.

Use Groq to:
### A. Improve OCR extraction
Return corrected structured JSON:
{
  name,
  dob,
  id_number,
  expiry_date,
  notes
}

### B. Fraud analysis
Return:
{
  risk_level,
  reasons,
  inconsistencies
}

Use model:
llama3-70b-8192

---

## 8. ADMIN PANEL (NO LOGIN)

Route:

- Edit:
